# Self Drink Bar System – テスト手順書（たたき台）
## 1. テスト概要
本テスト手順書は、Self Drink Bar System の利用フローおよび管理者画面の閲覧機能を確認するためのものです。
一般ユーザーはメニュー選択〜決済までの流れを、管理者は demo アカウントで管理画面の閲覧を行います。

## 2. テストに必要な情報
サイト URL：
https://selfdrinkbar-system-production.up.railway.app/ (selfdrinkbar-system-production.up.railway.app in Bing)

### 管理者ログイン情報（閲覧専用）

- ID：demo
- PW：demo
推奨ブラウザ：Google Chrome / Microsoft Edge

## 3. 一般ユーザー側テスト
### 3-1. メニュー選択
- トップページにアクセス
- 「メニュー」から任意のカテゴリ（コーヒー / 紅茶 / 緑茶 / ソフトドリンク）を選択
- 商品一覧が表示されることを確認
- 任意の商品を選択し「カートに追加」をクリック

### 3-2. カート確認
- 画面右上の「カート」をクリック
- カート内の商品名・数量・金額が正しく表示されていることを確認
- 数量変更・削除が正しく動作することを確認

### 3-3. 精算確認
- 「精算へ進む」をクリック
- 注文内容の最終確認画面が表示される
- 合計金額が正しいことを確認

### 3-4. 決済方法選択
- 「決済方法を選択する」をクリック

- 以下の選択肢が表示されることを確認

  - 現金
  - クレジットカード
  - paspo(交通系)
  - QRコード

- 任意の決済方法を選択

### 3-5. 決済
- 決済確認画面が表示される
- 「決済する」をクリック
- 決済完了メッセージが表示されることを確認

### 3-6. 完了
- 「完了」画面が表示される

トップページへ戻るリンクが表示されることを確認

## 4. 管理者画面（demo アカウント）テスト
※ demo アカウントは 閲覧のみ可能。更新・登録・削除は不可。

### 4-1. 管理者ログイン
- 画面右上の「管理者ログイン」をクリック
- ID：demo / PW：demo を入力
- 管理者ダッシュボードが表示されることを確認

### 4-2. 商品一覧の閲覧
- 「商品一覧」をクリック
- 商品名・カテゴリ・価格が一覧表示されることを確認
- 検索フォームで商品名検索ができることを確認

### 4-3. 商品登録画面の閲覧
- 「商品登録」をクリック
- 商品登録フォームが表示されることを確認
- demo アカウントでは登録できない旨のメッセージが表示されることを確認（POST 時）

### 4-4. CSV インポート / エクスポート画面の閲覧
- 「CSV インポート / エクスポート」をクリック
- CSV アップロードフォームおよびエクスポートボタンが表示されることを確認
- demo アカウントではインポート不可であることを確認（POST 時）

### 4-5. 購入履歴検索
- 「購入履歴検索」をクリック
- 日付・商品名・ユーザー名などの検索フォームが表示されることを確認
- 任意の条件で検索し、結果が表示されることを確認

### 4-6. パスワード変更画面の閲覧
- 「パスワード変更」をクリック
- パスワード変更フォームが表示されることを確認
- demo アカウントでは変更不可であることを確認（POST 時）

## 5. テスト完了
以上の項目を確認し、画面遷移・表示内容・閲覧制限が正しく動作していればテスト完了です。
